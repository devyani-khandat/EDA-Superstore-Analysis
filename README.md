# 📊 Superstore EDA & Business Intelligence Project
## 🔹 Objective
To perform Exploratory Data Analysis (EDA) and answer business questions using SQL, followed by creating a dashboard to visualize key insights.

---

## 📂 Dataset
* Superstore dataset (CSV)
* Contains information on:
  * Sales
  * Profit
  * Customers
  * Regions
  * Product categories
  * Order dates

---

## 🔍 Key Analysis Performed
### 1. Descriptive Statistics
* Summary statistics using Pandas
* Null value analysis
* Distribution analysis using histograms

### 2. SQL Business Questions
* Top 5 products by sales
* Highest revenue category
* Profit by region
* Top customers by sales
* Average order value

### 3. Multivariate Analysis
* Correlation heatmap
* Scatter plot (Sales vs Profit)
* Insights on relationships between variables

---

## 📊 Key Insights
* Technology category generates the highest sales
* West region contributes the highest profit
* Strong positive correlation between Sales and Profit
* Discounts negatively impact Profit
* Average order value is approximately ₹229.86

---

## 📈 Dashboard
The dashboard includes:
* Total Sales, Total Profit, Average Order Value
* Sales by Category
* Profit by Region
* Monthly Sales Trend

---

## 🛠 Tools Used
* Python (Pandas, Matplotlib, Seaborn)
* SQL (SQLite in Google Colab)
* Microsoft Excel (Dashboard creation)

---

## 🚀 Conclusion
This project demonstrates how data analysis and visualization can be used to derive actionable business insights.
